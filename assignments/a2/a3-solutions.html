  3. Assuming that a sample size of 50 is large enough for the CLT to kick in we would expect the sampling distribution to follow a normal distribution.
    The center of the sampling distribution is the population mean (\(\mu\)) -if you assumed that the distribution in table 1 to be a population then the mean in 2a is \(\mu\), if you assumed that the distribution in table to be a sample then the mean you calculate in 2a is ybar and because it is a large sample size we assume that ybar is an unbiased estimate of \(\mu\) - therefore the center of the distribution will be 137.5.
    The spread of the sampling distribution is standard error (the standard deviation of the means). If you treated distribution in table 1 as a population then SE = sigma/sqrt(n), if you treated the distribution as a sample then SE = sd/sqrt(n). (5 points)

# Confidence Interval using t procedure
#t score function
qt_ls <- function(p, df, mean, sd) qt(p = p, df = df) * sd + mean

n = 201

food_store_ybar <- 18.67
food_store_SE <- 24.95/sqrt(n)

# using the q function
food_store_CI <- qt_ls(p = c(0.025,0.975), df = n-1, mean = food_store_ybar, sd = food_store_SE)
# We can also use the plus/minus formula
# food_store_CI <- food_store_ybar + qt(p = c(0.025,0.975), df = n-1) * food_store_SE
food_store_CI <- round(food_store_CI, 2)

mass_merch_ybar <- 32.38
mass_merch_SE <- 33.37/sqrt(n)
mass_merch_CI <- qt_ls(p = c(0.025,0.975), df = n-1, mean = mass_merch_ybar, sd = mass_merch_SE)
mass_merch_CI <- round(mass_merch_CI, 2)

pharmacy_ybar <- 48.6
pharmacy_SE <- 35.62/sqrt(n)

SEybar.n20 <- sd(depths.n.20$alt) / sqrt(20)
SEybar.n20 <- round(SEybar.n20, 2)
###################################################

# 95% CI using plus/minus formula
ci95.n20.a <- ybar.n20 + c(-1.96,1.96) * SEybar.n20
ci95.n20.a <- round(ci95.n20.a, 2)

# 95% CI using qnorm
ci95.n20.b <- qnorm(p = c(0.025,0.975), mean = ybar.n20, sd = SEybar.n20)
ci95.n20.b <- round(ci95.n20.b, 2)

# 10000 bootstrap samples
bootstrap.n20 <- do(10000) * mean( ~ alt, data = resample(depths.n.20))

# 95% CI using bootstrap
CI_95 <- quantile(~ mean, data = bootstrap.n20, probs = c(0.025, 0.975))
CI_95 <- round(CI_95, 2)

(-3 points for not providing units for each CI)

  1. (5 points) 95% CI from plus/minus formula: [20.54, 39.66] meters.
  2. (5 points) 95% CI from qnorm function: [20.54, 39.66] meters.
  3. (5 points) 95% CI from 10000 bootstrap samples: [20.7, 39.25] meters.
compare_CI <- function(ybar, PM, QNORM, BOOT, 
                       col = c("#E41A1C","#377EB8","#4DAF4A")) {
  
  dt <- data.frame(type = c("plus_minus", "qnorm", "bootstrap"),
                   ybar = rep(ybar, 3),
                   low = c(PM[1], QNORM[1], BOOT[1]),
                   up = c(PM[2], QNORM[2], BOOT[2])
  )
  
  plot(dt$ybar, 1:nrow(dt), pch = 20, ylim = c(0, 5), 
       xlim = range(pretty(c(dt$low, dt$up))),
       xlab = 'Depth of ocean (m)', ylab = 'Confidence Interval Type',
       las = 1, cex.axis = 0.8, cex = 3)
  
  abline(v = 37, lty = 2, col = "black", lwd = 2)
  segments(x0 = dt$low, x1 = dt$up,
           y0 = 1:nrow(dt), lend = 1,
           col = col, lwd = 4)
  
  legend("topleft",
         legend = c(eval(substitute( expression(paste(mu," = ",37)))),
                    sprintf("plus/minus CI: [%.f, %.f]",PM[1], PM[2]),
                    sprintf("qnorm CI: [%.f, %.f]",QNORM[1], QNORM[2]),
                    sprintf("bootstrap CI: [%.f, %.f]",BOOT[1], BOOT[2])),
         lty = c(1, 1,1,1),
         col = c("black",col), lwd = 4)
}

compare_CI(ybar = ybar.n20, PM = ci95.n20.a, QNORM = ci95.n20.b, BOOT = CI_95)

  1. (10 points) In the 3 plotted confidence intervals, we can observe that 2 look identical and one looks very similar to the other 2. More specifically, the confidence interval calculated via the +/- formula (4a) and the confidence interval calculated via the qnorm function (4b) are identical. Additionally, both confidence intervals are symmetric in the sense that the width below and above the mean within the same confidence interval are the same value. However, this is one aspect that differentiates the bootstrap confidence interval (4c). The bootstrap is not entirely symmetrical on both sides of the mean within the interval. However, since the bootstrap interval is relatively similar to the intervals for +/- formula and the qnorm function, it can be suggested that the central limit theorum has kicked in and we can safelyperform inference based on the Normal distribution.
